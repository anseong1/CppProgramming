# 실습과제

## 1. 라이브러리는 헤더파일과 목적파일로 구성되어 있다. 헤더파일과 목적파일에 들어 있는 내용이 무엇인지 가가 설명 하시오.

헤더파일(.h)은 함수의 선언, 매크로 정의, 상수, 자료형 정의 등이 들어 있어 프로그램에서 해당 라이브러리의 기능을 사용할 수 있도록 알려주는 역할을 하고
목적파일(.obj)은 실제로 실행될 기계어가 들어 있는 파일로, 컴파일된 결과가 저장되어 있다.

---

## 2. 함수의 선언, 정의, 호출은 각각 어디에 존재하는가?
함수의 선언은 헤더파일에 존재하여 함수의 이름과 형태를 미리 알려주고, 정의는 목적파일에 존재하여 함수의 실제 동작이 구현되어 있으며, 호출은 사용자가 작성한 소스파일(.cpp)에서 이루어진다.
