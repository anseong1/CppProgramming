# 실습과제

## 1. 구조체와 클래스의 차이점을 설명하라.
구조체와 클래스의 차이점으로는 기본 접근 지정자가 다르다는 것이다. `struct`는 
멤버 변수와 함수가 기본적으로 public이고, `class`는 기본적으로 private 이다. 상속할 때도 마찬가지로 `struct`는 기본이 public 상속, `class`는 private 상속이다.

---

## 2. 클래스와 객체의 차이점을 설명하라.
클래스는 객체를 만들기 위한 설계도 또는 틀이며, 변수와 함수(멤버 변수, 멤버 함수)를 정의한 것이다. 객체는 이 클래스를 기반으로 실제로 메모리에 생성된 실체이다. 
정리하면, 클래스는 객체를 만들기 위한 틀이고 객체는 그 틀에서 만들어진 
실제 사용 가능한 데이터이다.

---

## 3. sizeof(클래스명)의 결과값을 출력해보고 결과를 설명하라.
`sizeof`는 객체나 자료형의 메모리 크기를 바이트 단위로 구하는 연산자이다.
실습문제2를 예시로 들어 맨 아래의 사진과 같이 `sizeof` 연산자를 넣어서 코드를 짜보면
```
삼각형의 면적은 25
height의 크기: 4 byte
width의 크기: 4 byte
Triangle 객체 크기: 8 byte
```
위와 같이 결과값이 나오게 된다. 
`sizeof(t.height)`는 변수 크기 / `sizeof(Triangle)`는 객체 전체 크기 의미한다.
여기서 `getArea()`, `sizeof` 같은 멤버 함수는 객체 메모리에 포함되지 않는다.
