# 실습과제

## 소스코드설명

```
#include<iostream>
```
- 입출력을 위한 헤더파일이며 `cout`, `endl` 등을 사용하기 위해 필수적

```
using namespace std;
```
- `std` namespace 선언으로 `cout`,`endl` 사용시 앞에 std:: 생략 가능

```
int main(void)
```
- 메인함수 시작점

```
for (int i = 0; i < 4; i++)
```
- 열을 출력하기 위한 반복문 `j <= i`를 사용하여 현재 행의 갯수까지 `*`을 출력

```
cout << endl;
```
