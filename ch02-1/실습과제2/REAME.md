# 실습과제

## 코드설명

```
#include<iostream>
```
- 입출력을 위한 헤더파일이며 cout, endl 등을 사용하기 위해 필수적

```
int main(void)
```
- 메인함수 시작점

```
std::cout << "이름 : 홍길동" << std::endl;
std::cout << "주소 : 군산시 대학로 558" << std::endl;
std::cout << "학번 : 201012" << std::endl;
std::cout << "차종 : 벤츠" << std::endl;
```
- std::cout은 화면에 텍스트를 출력하며, std::endl은 줄바꿈을 의미

```
return 0;
```
- 프로그램 종료를 알리는 반환값
