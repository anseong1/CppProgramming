# 실습과제

## 1. 전처리 또는 선행처리 단계를 설명하라.

 컴파일러가 코드를 번역하기 전에 전처리기가 `#` 으로 시작하는 전처리 지시문을 처리하는 과정이다.

| 지시어 | 뜻 |
|--------|------|
| `#include` | 헤더 파일 내용을 소스에 삽입 |
| `#define` | 매크로 상수/함수 정의 |
| `#ifdef / #endif` | 조건부 컴파일 |

## 3. 프로그램을 링크할때 이름 충돌이 발생하는 경우를 설명해보라.

여러 라이브러리나 파일을 링크할 때 같은 이름의 함수나 변수가 중복 정의되면  
링커가 어느 것을 사용할지 판단하지 못해 오류가 발생한다.


---


## 4. 스트림이란 무엇인가?

데이터가 입력되거나 출력되면서 흐르는 통로를 의미하며, 입력 스트림은 데이터가 프로그램으로 들어오는 흐름이고 
출력 스트림은 데이터가 프로그램 밖으로 나가는 흐름이다.


---


## 5. 표준 출력 스트림을 정의할 때 사용하는 C언어 자료형은?

 `FILE` 자료형을 사용한다.

## 6. 표준 출력 스트림을 정의할 때 사용하는 C++언어 자료형은?

`std` namespace 에 선언된 `ostream` 자료형을 사용한다.
